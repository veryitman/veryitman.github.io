简介

本文分享在 MacOS 上面通过 pkg 的方式安装 GO 编程环境, 以及编译运行第一个使用 GOLang 编写的程序.

其他关于 GO 的安装方式, 如使用 Homebrew 安装或者使用源码安装的方式, 请自行尝试.

卸载 GO

如果你想卸载 GO 环境, 只需要三个步骤.

1.删除安装目录

2.删除 go 文件

3.删除环境变量

编辑 etc, 删除对应添加的 go 环境变量即可.
